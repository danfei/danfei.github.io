如果截图选择region的时候，出现图像变暗的情况
可以用如下方法恢复
右键点击menu
Preferences -> Compositor -> Edit Composition Settings
或者编辑 ~/.config/compton.conf
在这一行

shadow-exclude = [ “i:e:Conky” ];
后再加一行

保存重启compositor
然后就一切就OK了
